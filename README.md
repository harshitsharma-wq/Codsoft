# CODSOFT C++ Programming Internship

This repository contains the C++ projects completed during my CodSoft C++ Programming Internship.

## Projects

### 1. Number Guessing Game
A console-based game where the user guesses a randomly generated number.

**Features**
- Random number generation
- Unlimited attempts
- High/Low hints
- Attempt counter
- Play again option

---

### 2. Simple Calculator
A console-based calculator that performs basic arithmetic operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Repeat calculations

---

### 3. Tic Tac Toe Game
A two-player console-based Tic Tac Toe game.

**Features**
- 3×3 game board
- Two-player mode
- Win detection
- Draw detection
- Input validation
- Play again option

---

## Technologies Used

- C++
